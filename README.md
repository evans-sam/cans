## Description

A simple api to manage a cans service

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Open API
<a href="http://localhost:3000/openAPI" target="_blank">Click here to use the open api interface</a>
```
http://localhost:3000/openAPI
```

## Routes
```
 GET: /api/resolve/:name
 
 POST: /api
 GET: /api
 GET: /api/:id
 PATCH: /api/:id
 DELETE: /api/:id
```
